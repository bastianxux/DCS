# 流式框架实验

### 实验目的
 * 本实验基于流式框架Storm、Spark Streaming、Flink ，对比分析其在并行处理能力和特性上的表现，并探索Kafka Streaming流式数据处理框架的特性。

### 实验方法
 * 通过docker搭建不同框架的集群，使用 socket 进行数据传输模拟，将评估各框架在接收实时数据并进行 k-NN 计算的表现，通过参数设定，测试吞吐量、延迟等指标，根据框架特性对结果进行分析解释，并对框架之间进行对比总结。
